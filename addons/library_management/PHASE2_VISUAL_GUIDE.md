## State Management

### Reading List State
```javascript
// Stored in browser
localStorage: {
  "readingList": [
    { "id": 5, "title": "Python Guide", "authors": "Author A" },
    { "id": 12, "title": "Web Dev", "authors": "Author B" }
  ]
}

// Loaded on page load
// Updated on heart click
// Persists across sessions
```

### Filter State
```javascript
// Active filters (in memory)
{
  searchTerm: "python",
  authorFilter: "15",
  publisherFilter: "",
  sortOrder: "title-asc",
  activeCategory: "all"
}

// Resets on clear
// Updates on interaction
// Does not persist
```
